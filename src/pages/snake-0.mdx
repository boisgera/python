## The Hard Way

Useful if you want:

  - to develop Python projects other than Pyxel games,

  - to work locally on your computer and not depend on "the cloud",

  - to benefit from a more feature-rich development environment.


### Python

You can get a recent release of Python from the [official Python website],
or use the [miniconda][miniconda] or [anaconda][anaconda] package
managers instead ; they are very handy in my opinion to manage 
multiple Python projects.

[official Python website]: https://www.python.org/downloads/
[miniconda]: https://docs.conda.io/projects/miniconda/en/latest/miniconda-install.html
[anaconda]: https://www.anaconda.com/download/

### Visual Studio Code

  - Install [Visual Studio Code][VS-Code], then

  - Install the Microsoft [Python extension for Visual Studio Code][Python-VS-Code].

[VS-Code]: https://code.visualstudio.com
[Python-VS-Code]: https://marketplace.visualstudio.com/items?itemName=ms-python.python

  - (Optional) Enable [Github Copilot on Github]. 
    (This is free for students registered into their [Global Campus] program ; 
    alternatively, you may opt for a 30 free trial.)     
    Then install the [GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot)
    extension for Visual Studio Code.

[Github Copilot on Github]: https://docs.github.com/en/copilot/quickstart
[Global Campus]: https://education.github.com/

### Install Pyxel

Use `pip` to install Pyxel (there is no conda package for it).

```bash
pip install pyxel
```

## Troubleshooting

Common issues that you may have with "the hard way":

<Admonition type="warning" title="Pyxel is not found" open={debug}>
You have installed Pyxel with pip but Visual Studio Code cannot find it?
It is likely not using the same Python environment as the one
you used to install Pyxel.

  - In your terminal, type:
  
    ```bash
    python -c "import sys; print(sys.executable)"
    ```
 
    to know to which Python interpreter `python` refers to. 
   
  - In Visual Studio Code, open the command palette with `Ctrl+Shift+P` 
    and type `Python: Select Interpreter` to select this Python interpreter.

  - Install explicitly pyxel for this interpreter with:

    ```bash
    python -m pip install pyxel
    ```

    (your `pip` executable may refer to another Python interpreter.)

</Admonition>

<Admonition type="warning" title="undefined symbol: SDL_Metal_DestroyView" open={debug}>

[@mgmalheiros] has analyzed this issue on [GitHub][issue-SDL_Metal_DestroyView]. 
A workaround is to use an older version of Pyxel:

```bash
pip install pyxel==1.9.7
```


</Admonition>
